```
NAME
	asn_xor - logical xor of two arbitrary size numbers

SYNOPSIS
	string asn_xor(string a, string b)


DESCRIPTION
	Compute the bit-wise logical xor of a and b (sign extended).  The
	result will be as long as the longest argument.

```
